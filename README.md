[![FreeBSD](https://img.shields.io/badge/FreeBSD-Port-red)](https://www.freebsd.org/ports/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Temporal](https://img.shields.io/badge/Temporal-CLI-blue)](https://github.com/temporalio/cli)

A FreeBSD port for the [Temporal CLI](https://github.com/temporalio/cli) - the official command-line tool for interacting with Temporal Server.

## About Temporal CLI

Temporal CLI is a command-line interface for running Temporal Server and interacting with Workflows, Activities, Namespaces, and other parts of the Temporal workflow engine.

**Key Features:**
- 🖥️ Server management and administration
- 🔄 Workflow and activity interaction  
- 📁 Namespace management
- ⚡ Task queue operations
- 📊 Batch operations and search capabilities
- 🔧 Development server for local testing

## Quick Installation

### Method 1: Using the Port Files

```bash
# Clone this repository
git clone https://github.com/seunets/temporal-cli-freebsd-port.git
cd temporal-cli-freebsd-port

# Copy to ports tree
sudo mkdir -p /usr/ports/devel/temporal-cli
sudo cp Makefile pkg-descr pkg-plist distinfo /usr/ports/devel/temporal-cli/

# Build and install
cd /usr/ports/devel/temporal-cli
sudo make makesum
sudo make install clean
```

### Method 2: Manual Build

```bash
# Install Go if not already installed
sudo pkg install go121

# Download and build Temporal CLI directly
go install github.com/temporalio/cli/cmd/temporal@latest
```

## Usage Examples

```bash
# Start Temporal server in development mode
temporal server start-dev

# In another terminal, interact with Temporal
temporal namespace list
temporal workflow list

# Create a namespace
temporal namespace create my-namespace

# See all available commands
temporal --help
```

## Port Structure

```

├── distinfo           # Checksums (generated by make makesum)
```
